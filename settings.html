Where to find your credentials
📱
WhatsApp — Phone Number ID & Business Account ID
Go to Meta Business Suite → WhatsApp → API Setup. You'll find the Phone Number ID and WABA ID on that page. Create a System User token with whatsapp_business_messaging and whatsapp_business_management permissions — this becomes your Permanent Access Token.
📸
Instagram — Page ID & Access Token
Go to Meta for Developers → your app → Instagram → Instagram Basic Display or Instagram Graph API. The Page ID is your connected Facebook Page ID. Generate a Page Access Token via the Graph API Explorer with the instagram_manage_messages permission.
📘
Facebook — Page ID & Access Token
Go to Meta for Developers → your app → Messenger → Access Tokens. Select your Page to generate a Page Access Token. The Page ID is visible in your Facebook Page's About section → Page ID.
💬
SMS — Twilio Account SID & Auth Token
Go to Twilio Console → Dashboard. Your Account SID and Auth Token are shown at the top. Buy or port a phone number from Twilio → Phone Numbers, and enter it in E.164 format (e.g. +49301234567).
